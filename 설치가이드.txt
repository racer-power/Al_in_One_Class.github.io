  올인원 학급 운영 웹앱 - 설치 가이드
═══════════════════════════════════════════════════

■ 파일 구성
  ├── Code.gs        ← 백엔드 (서버 로직)
  ├── Env.gs         ← 환경 변수 (Script Properties)
  ├── Index.html     ← 프론트엔드 (화면 구조)
  ├── Styles.html    ← 프론트엔드 (CSS 스타일)
  ├── Client.html    ← 프론트엔드 (JavaScript)
  └── appsscript.json

■ 1단계: Google 스프레드시트 만들기
  1. Google Drive에서 새 Google 스프레드시트를 만듭니다.
  2. 확장 프로그램 → Apps Script 를 클릭합니다.

■ 2단계: 코드 붙여넣기
  1. 기본 Code.gs 내용을 삭제하고, Code.gs 파일 내용을 붙여넣습니다.
  2. + 버튼 → HTML 로 Index.html, Styles.html, Client.html 을 각각 추가합니다.
     (파일명은 반드시 Index, Styles, Client — 확장자 없이)
  3. 프로젝트 설정 → appsscript.json 표시 → appsscript.json 내용 붙여넣기

■ 3단계: 초기 설정 실행
  1. Apps Script 편집기에서 함수 선택: setupSheets
  2. ▶ 실행 → Google 계정 권한 허용
  3. 자동으로 3개 시트(학생명단, 출결로그, 공지사항)와
     Google Drive 폴더(학급운영_서명보관)가 생성됩니다.

■ 3-1단계: 환경 변수 등록 (선택)
  1. Env.gs 파일의 setupEnvironment() 함수 안 env 값을 수정
  2. 함수 선택: setupEnvironment → ▶ 실행
  3. 또는 Apps Script → 프로젝트 설정 → Script properties 에서 직접 등록
  4. 로컬 참고용: .env.example 을 복사하여 .env 생성 (Git 업로드 제외)

■ 4단계: 학생 데이터 입력
  [학생명단] 시트
    A열: 학번  |  B열: 이름
    예) 101  |  김민수

  [공지사항] 시트
    A1, A2, A3... 에 공지 작성 (가장 아래 줄이 학생에게 표시)

■ 5단계: 웹앱 배포
  1. 배포 → 새 배포
  2. 유형: 웹 앱
  3. 실행 주체: 나
  4. 액세스: 모든 사용자 (또는 조직 내 사용자)
  5. 배포 → URL 복사 → 학생들에게 공유

■ 주요 기능 요약
  ✓ 학번 입력 → 학생 확인 → 환영 메시지
  ✓ 공지사항 표시 (가장 최신)
  ✓ 이번 달 달력 + 출석일 💖 표시
  ✓ 출석하기 / 퇴실하기 동적 버튼
  ✓ 퇴실 시 배움일지 + 서명 → Drive 저장 + 시트 기록
  ✓ 시트 서명 열에 =IMAGE() 수식으로 이미지 직접 표시
  ✓ 모든 통신 시 로딩 스피너 표시

■ 문제 해결
  - "시트를 찾을 수 없습니다" → setupSheets 함수 실행
  - 서명 이미지가 안 보임 → Drive 파일 공유가 자동 설정됨 (1.5초 대기 포함)
  - 권한 오류 → 배포 시 "나"로 실행, Drive/Sheets 권한 재승인
